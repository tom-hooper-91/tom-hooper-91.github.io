Positioning in CSS

What are the differences between relative, absolute, and fixed positioning?

Relative

Relative position acts exactly the same as static positioning (which is the default position for every element) except it allows you to move the element independently of where it would normally be in the document flow. You can do this by setting the top, right, bottom and left properties. This is pretty useful because a relative element won't affect any of the surrounding elements when you move it this way!
