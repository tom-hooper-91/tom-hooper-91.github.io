Positioning in CSS

What are the differences between relative, absolute, and fixed positioning?

This is a quick explanation on how to position elements using CSS. An element is a building block that web developers use to build a website, think of it like a brick being used to build a house. To make the examples easier to visualize I have used a border to make a white box around the elements I talk about in my examples, think of the white box like a wall or section of the house with multiple bricks inside it. A property is just another way of telling a brick how to behave.

Relative

Relative position acts exactly the same as static position (which is the default for every element) except it allows you to move the element independently of where it would normally be in the document flow (or wall). You can do this by setting the top, right, bottom and left properties. This is pretty useful because a relative element won't affect any of the surrounding elements when you move it this way!

Here I have used the property top to move the relative element down 10px, you can see the other static elements haven't moved.

Absolute

Absolute position completely removes the element from the document flow, this means the other surrounding elements competely ignore the absolute element. If you move the element using top, right, bottom and left properties it will move in relation to it's parent container which is positioned. A positioned element is one that has any position other than static. If none of its parent elements are positioned it will fall back to the html document body and stay in the same position even if scrolling.

Now I have used the propery top to move the absolute div 10px from the top of it's positioned parent container. I had to chang the parent container to relative so that it is positioned, otherwise it would position itself 10px from the top of the document body.

Fixed

Fixed position automatically falls back to the document body and sticks there even when scrolling. This is the same behaviour an absolute positioned element would have if none of its parent containers were positioned.

I have positioned the fixed element using the bottom property to position the element 10px from the bottom of the page. You may have already noticed it hovering around.
